El siguiente repositorio tiene la función de compartir con el profesorado y alumnado de la asignatura de circuitos eléctricos de física, las funciones, algoritmos, desarrollos y teoría para la realización de las prácticas de laboratorio Carga y Descarga del condensador y Estudio en Frecuencias del RC.

Se pretende distribuir las funciones desarrolladas para salvar la barrera del lenguaje de programación de Python y así, que el alumnado, sea capaz de graficar y calcular transformadas discretas de Fourier de una manera mucho más sencilla. El objetivo último es este, simplificar la tarea de programación al alumnado que usará estas funciones como "ladrillos" para construir la práctica de laboratorio.

En este readme.txt se explica la disposición del repositorio y los distintos contenidos que se encontrarán.

Primero se insta al alumnado a mirar el TFG completo, bajo el nombre Memoria TFG.pdf, aquí se explica detalladamente todos los pasos, conceptos y objetivos desarrollados.

En Desarrollo completo (profesor).ipynb, hemos subido todo el proceso de este repositorio, desde la creación de las funciones y algoritmos, hasta las llamadas a las mismas y los comentarios.

En Funciones (alumnos).ipynb, se pretende distribuir solo las funciones que los alumnos usaran como piezas de lego o ladrillos, sin llamadas a las funciones, el alumnado usará este documento

En Metodologia.ipynb se hace un desarrollo de los conceptos teóricos muy detallado

En Python_Bibliotecas.ipynb hay una explicación de como usar Python, así como una guía de las bibliotecas utilizadas, matplotlib y numpy.
